Gulnaz's Data Science Projects Portfolio

Project 1: Evaluating the Efficacy of Machine Learning Models in Time-series Forecasting

This research aims to identify the most competent machine learning model for effectively anticipating Bitcoin's future price movements. To address this need comprehensively, the study utilizes three leading machine learning techniques: the Support Vector Machine (SVM), the Long Short Term Memory (LSTM) neural network, and the predictive capacities of Facebook's Prophet model.

Project 2: Proportional Species Richness Analysis

This project undertook a comprehensive analysis of proportional species richness data using R, employing tools such as histograms for data distribution assessment and t-tests for hypothesis testing between periods. Additionally, data preparation involved categorizing columns and introducing new variables, while exploratory data analysis included pairwise correlation analysis to uncover interrelationships among species variables.

Project 3: Natural Language Processing with Disaster Tweets

The project aims to distinguish tweets discussing real disasters from those that do not, utilizing fundamental NLP techniques and employing a Term Frequency-Inverse Document Frequency (TF-IDF) approach. A basic random forest model is implemented to conduct tweet classification based on extracted features.
